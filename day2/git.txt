# Git commands
=> Local and Remote Repository (Repo)

1. git config --global user.name "username"
2. git config --global user.email "email"
3. git config --global user.name
4. git config --global user.email


5. git init
    => Initializes a local repo

6. git add .
    => Adds all the changes to git
7. git commit -m "commit message"
    => Commits the added changes
8. git remote add origin <ssh-link>

9. git remote -v
    => Lists all the remote repo ssh link
